# MongoDB-API

# Clone this repository
``` git clone https://github.com/7Biscuits/MongoDB-API ```

# Change your directory to the repo
``` cd MongoDB-API ```

# Install necessary modules
``` yarn ```
<br>or<br>
``` npm ```

# Run app.js
``` node app.js ```

# Open your browser and paste this url
``` http://localhost:3000/tasks ```
